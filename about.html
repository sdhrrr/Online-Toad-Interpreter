About Toad Programming Language

Toad is a dynamically typed, multi-paradigm programming language that I developed. It is designed to be simple and easy to use, making it suitable for learning programming concepts as well as for scripting tasks.

Key Features

  • Simple Syntax: Toad offers a clean and straightforward syntax that is easy to read and write.
  • Dynamic Typing: Variables in Toad do not require explicit type declarations, allowing for flexibility in coding.
  • Multi-Paradigm Support: Supports imperative and functional programming paradigms.
  • Interpreted Language: Code is executed directly by the Toad interpreter without the need for compilation.

Why Toad?

The motivation behind creating Toad was to develop a language that is both educational and practical. It serves as a project to explore the complexities of language design and implementation.

Getting Started with Toad

Here's how you can start writing code in Toad using the online interpreter:

  1. Navigate to the Home page of the Toad interpreter.
  2. Write your Toad code in the code editor provided.
  3. Click the Run button or press Ctrl + Enter to execute your code.
  4. View the output in the output area.

Language Syntax and Examples

Toad's syntax is inspired by languages like C and JavaScript. Here are some examples:

Variables and Data Types

// Variables
var x = 10;
var name = "Toad";

// Printing
print(x);
print(name);

Functions

function greet(name) {
    print("Hello, " + name + "!");
}

greet("Alice");

Control Structures

// If-else
var age = 18;
if (age >= 18) {
    print("You are an adult.");
} else {
    print("You are a minor.");
}

// Loops
for (var i = 0; i < 5; i = i + 1) {
    print(i);
}
